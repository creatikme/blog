# React node blog
This is a blog built using [React.js](https://reactjs.org), [Redux](https://redux.js.org), and [Node.js](https://nodejs.org).

# Technologies

1. [React.js](https://reactjs.org)
1. [Redux](https://redux.js.org)
1. [create-react-app](https://github.com/facebook/create-react-app)
1. [Express.js](https://expressjs.com)
1. [Node.js](https://nodejs.org)
1. [MongoDB](https://mongodb.com)

### Features
- Write a story
- Social Sign-in
- View a story
- View all stories
- Follow a user
- Clap a story
